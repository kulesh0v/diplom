\bigskip

\noindent КЛЮЧЕВЫЕ СЛОВА ОПТИМАЛЬНОЕ УПРАВЛЕНИЕ, УПРАВЛЕНИЕ ПО ПРОГНОЗИРУЮЩЕЙ МОДЕЛИ, MPC


\bigskip

Объектом исследования курсовой работы является управление по прогнозирующей модели.

Цель работы - анализ схемы управления по прогнозирующей модели.

Работа состоит из трёх глав. В первой главе определяются основные понятия в области управления с прогнозирующей моделью. Во второй главе рассматриваются способы решения прогнозирующих задач. В третьей главе показываются результаты численных эксперементов и проводится их анализ.
