\chapter*{Введение}
\addcontentsline {toc}{chapter}{ВВЕДЕНИЕ}
Методы, ориентированные на данные, в последние годы всё чаще используются для анализа и управления систем, но лишь немногие такие методы дают теоретические гарантии на устойчивость или удовлетворение системных ограничений. Метод управления, который хорошо подходит для достижения этих целей, -- это управление по прогнозирующей модели (MPC), которое может обрабатывать нелинейную динамику системы, жесткие ограничения на состояние и входные и выходные сигналы, а также учитывает критерии эффективности \cite{Berberich}.


Данные методы сосредоточены на повторяющемся онлайн-решении задачи оптимизации по прогнозируемым системным траекториям. MPC-регулятор данного метода обычно полагается на эмпирическую модель процесса для того, чтобы предсказать дальнейшее его поведение, основываясь на предыдущих значениях переменных состояния. Но также существует привлекательная альтернатива реализации MPC-регулятора основывающегося непосредственно на измеренных данных без предварительного знания точной модели.

В главе \ref{chap1} приведены основные сведения из теории у управления по прогнозирующей модели, а также рассматриваются задачи оптимального управления и их классификация.
 В главе \ref{chap2} исследуются методы решения \cite{Berberich} простейшей прогнозирующей задачи на основе данных и задачи с терминальными ограничениями-равенствами.
 В главе \ref{chap3} показано решение \cite{Berberich} проблемы зашумленных данных с помощью робастной схемы MPC.
В главе \ref{chap4} применены схемы MPC из глав \ref{chap2} и \ref{chap3}, рассмотрены программные реализации данных схем и приведены полученные результаты.
