\chapter*{Введение}
\addcontentsline {toc}{chapter}{ВВЕДЕНИЕ}
Методы, ориентированные на данные, в последние годы всё чаще используются для анализа и управления систем, но лишь немногие такие методы дают теоретические гарантии на устойчивость или удовлетворение системных ограничений. Метод управления, который хорошо подходит для достижения этих целей, -- это управление по прогнозирующей модели (MPC), которое может обрабатывать нелинейную динамику системы, жесткие ограничения на состояние и входные и выходные сигналы, а также учитывает критерии эффективности.


Данные методы сосредоточены на повторяющемся онлайн-решении задачи оптимизации по прогнозируемым системным траекториям. MPC-регулятор данного метода обычно полагается на эмпирическую модель процесса для того, чтобы предсказать дальнейшее его поведение, основываясь на предыдущих значениях переменных состояния. Но также существует привлекательная альтернатива реализации MPC-регулятора основывающегося непосредственно на измеренных данных без предварительного знания точной модели.

В главе \ref{chap1} приведены основные сведения из теории у управления по прогнозирующей модели, а также рассматриваются задачи оптимального управления и их классификация.
 В главе \ref{chap2} исследуются методы решения \cite{Berberich} простейшей прогнозирующей задачи на основе данных и задачи с терминальными ограничениями-равенствами.
 В главе \ref{chap3} показано решение \cite{Berberich} проблемы зашумленных данных с помощью робастной схемы MPC.
В главе \ref{chap4} применены схемы MPC из глав \ref{chap2} и \ref{chap3}, рассмотрены программные реализации данных схем и приведены полученные результаты.

\chapter*{ПЕРЕЧЕНЬ УСЛОВНЫХ ОБОЗНАЧЕНИЙ}
\addcontentsline {toc}{chapter}{ПЕРЕЧЕНЬ УСЛОВНЫХ ОБОЗНАЧЕНИЙ}

$\mathbb{I}_{[a,b]}$ -- множество целых чисел на отрезке $[a,b]$.

Для вектора $x$ и положительно определённой матрицы $P=P^T>0$ запишем $\|x\|_P=\sqrt{x^TPx}$.

$
H_L(x)=
\begin{bmatrix}
	x_0 && x_1 && \dots && x_{N-L} \\
	x_1 && x_2 && \dots && x_{N-L+1} \\
	\vdots && && \ddots && \vdots \\
	x_{L-1} && x_L && \dots && x_{N-1}
\end{bmatrix}
$ --  матрица Ганкеля, составленная из последовательности $\{x_k\}^{N-1}_{k=0}$  


Определим максимальное и минимальное собственные значения матрицы P
$\lambda_{\min}(P)$ и $\lambda_{\max}(P)$. Для матриц $P_1=P_1^T, P_2=P^T_2$ запишем, 	
\begin{displaymath}
	\lambda_{\min}(P_1,P_2)=\min\{\lambda_{\min}(P_1),\lambda_{\min}(P_2)\},
\end{displaymath}
\begin{displaymath}
	\lambda_{\max}(P_1,P_2)=\max\{\lambda_{\max}(P_1),\lambda_{\max}(P_2)\}.
\end{displaymath}

 $\|x\|_2,\|x\|_1$, $\|x\|_\infty$ -- Евклидовы, $\ell_1$ и $\ell_\infty$ нормы $x$, соответственно.
